A WordPres WooCommerce plugin to add Google reCAPTCHA v.2 to WooCommerce login-register and lost password forms.

== Description ==

Google reCAPTCHA v.2 adds to the following WooCommerce forms:

- My Account (login - register)
- Lost your password

If reCAPTCHA fails - forms do not proceed.

== Installation ==

Install the plugin as usual.

After the plugin activation get Google reCAPTHCA site key and secret key from https://www.google.com/recaptcha

Copy them and paste on the plugin settings page on your WordPress site. Save settings.
